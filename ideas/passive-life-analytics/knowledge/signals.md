# Signals already recorded

The premise: almost no new tracking is needed. A rough inventory of what a phone
already holds.

| Signal               | Source                | Already on device? |
| -------------------- | --------------------- | ------------------ |
| Places & movement    | Location history      | Yes                |
| Sleep & steps        | Health sensors        | Yes                |
| Communication rhythm | Call/message metadata | Yes                |
| Focus & screen time  | Usage stats           | Yes                |
| Photos as memory     | Camera-roll timestamps| Yes                |

> **Constraint:** if a signal needs new hardware or manual logging, it is out of
> scope for the first version.
